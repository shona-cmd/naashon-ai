# Security Policy

## Reporting Security Vulnerabilities

If you discover a security vulnerability in the AI Coding Assistant, please email security@naashonai.dev instead of using the issue tracker.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We take security seriously and will respond promptly to security reports.

## Security Best Practices

When using this extension:

1. **API Keys**: Keep your API keys confidential. Never commit them to version control.
2. **Code Scanning**: Review AI-generated code before using it in production
3. **Permissions**: The extension only accesses the current editor and VS Code configuration
4. **Dependencies**: We regularly update dependencies to patch security vulnerabilities

## Supported Versions

- Latest version: Full support
- Previous version: Security fixes only
- Older versions: No support

## Security Updates

Security updates will be released as needed and published to:
- VS Code Marketplace
- Open VSX Registry
- GitHub Releases

Subscribe to releases to stay informed about security updates.
